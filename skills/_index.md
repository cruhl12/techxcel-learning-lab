# Skills Index

A library of reusable SKILL.md files built during the TechXcel Learning Lab training program.

**How to add a skill:** Copy `_template.md`, rename it `skill-[kebab-case-name].md`, complete it, and add it to this index.

**Standard:** Every skill must be detailed enough for someone else to follow without asking you questions.
